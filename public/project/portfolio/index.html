* design motive *

우주는 모든 것을 알 수없는 미지의 세계라는 특징이 상당히 매력적으로 다가오는 소재입니다.
우주라는 방대한 컨셉을 통해 무한한 가능성을 탐구하며 스스로를 한정 짓지 않고 계속해서
확장해나가는 디자이너로서의 정체성을 표현하고자 포토폴리오의 컨셉을 우주로 설정하였습니다.

* consept keyword *

* project goal *

우주라는 무한한 공간을 탐구하는 여정을 통해 사용자의 경험과 가치를 넓히는 디자인을 창출해나가는 것을 목표로합니다.

단순 작업물 나열이 아닌 우주라는 컨셉을 역동적인 요소와 과하지 않은 모던한 레이아웃에 녹아내려 보는이에게 본인의 장점과 페이지의 컨셉이 잘 연결될 수 있도록 디자인 하였습니다.

* design system *

# 70BDF5, # FFD628

하늘색은 차분함과 신비로움, 노란색은 태양과 별빛을 연상시키며 우주 속 무한한 가능성을 나타내었습니다.

두 색상의 조합은 대비되는 색상으로 전체적인 균형감을 주고, 에너지와 신뢰감을 표현하였습니다. 또한 어두운 배경 속에 강한 대비를 만들어 도전과 희망적인 메세지를 전달하고자 하였습니다.

* typography *

Noto Sans (본고딕)

포트폴리오 내 본문과 내용은 사용자가 컨텐츠를 편안하게 읽을 수 있도록 가독성을 높
이고, 콘텐츠의 깔끔한 표현을 위해 Noto Sans(본고딕)을 사용하였습니다.

Abhaya Libre", serif

포인트를 줄 수 있는 요소로 세리프(Serif) 서체 특유의 정제된 느낌이 가독성을 해치지 않고 고급스러움과 신뢰감을 줄 수 있어 Abhaya Libre", serif를 사용하였습니다.
